# 💼 Homepage Project

A responsive and accessible **personal homepage** built with HTML and CSS, with a small amount of JavaScript for interactivity.
The main goal of this project was to **practice accessibility, responsive design, and modern CSS layout techniques**, while keeping the code clean and semantic.

---

## 🚀 Live Demo

🌍 **Visit the website:**  
👉 https://mcurb.github.io/homepage-TOP-project/

---

## ✨ Features

- ♿ **Accessible design**
  - Semantic HTML structure
  - WAI-ARIA attributes where appropriate
  - Built following WCAG guidelines

- 📱 **Fully responsive layout**
  - Mobile-first approach
  - Optimized for mobile, tablet, and desktop
  - Media queries for layout adjustments
  - Natural HTML responsiveness
  - Resolution switching and art direction for images

- 🎨 **Modern CSS layout**
  - CSS Grid with `auto-fit` and `minmax()`
  - Flexible and scalable sections

- 🎬 **Animations & transitions**
  - Subtle transitions for better UX
  - CSS animations for visual feedback
  - Respects `prefers-reduced-motion` for accessibility

- 🧠 **Clean structure**
  - Organized sections
  - Clear visual hierarchy
  - Focus on readability and usability

## 💻 Tech Stack

- HTML5 (Semantic markup)
- CSS3 (Grid, Media Queries, Animations)
- JavaScript (Minimal interactivity)

---

## 🧠 What I Practiced

- Implementing **accessible web design** using WAI-ARIA and semantic elements
- Applying **WCAG principles** in real-world layout decisions
- Building fully responsive layouts with:
  - Media queries
  - Grid `auto-fit` + `minmax()`
  - Resolution switching
  - Art direction techniques
- Respecting user preferences with `prefers-reduced-motion`
- Creating smooth, meaningful animations without harming usability
