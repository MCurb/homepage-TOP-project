About me

I’m learning in public, one project at a time. As a self-taught web developer, I focus on building simple, functional apps while sharpening my understanding of JavaScript and clean architecture. I’m not an expert yet — but I’m consistent, curious, and committed to improving every single day.

My Work

  • Weather App

    A weather app built with the Visual Crossing API to practice working with APIs, promises, and async/await in JavaScript.

  • To-do List

    A simple and fun web app to help you organize your tasks, set due dates, and stay on top of your day!

  • Battleship Game

    The main goal of this project was to practice testing, architecture, and game state management, while keeping the UI simple and functional.

  • Tic Tac Toe

    A simple Tic Tac Toe game built to practice the IIFE module pattern and keep the global scope clean. The project focuses on organizing game logic into private modules and exposing only the necessary methods.

  • Calculator App

    A basic calculator built to practice conditionals, DOM manipulation, and core JavaScript fundamentals. The project reinforces logical thinking and dynamic UI updates based on user input.

  • Etch-A-Sketch

    An interactive Etch-A-Sketch app built to strengthen DOM manipulation, event listeners, and loop logic. The project dynamically generates a grid and updates styles in response to user interaction.
